# sample_repo2

A simple Python calculator module.

## Structure

```
sample_repo2/
├── src/
│   └── calculator.py
├── test_calculator.py      # test file in root, no tests/ folder
├── requirements.txt
├── setup.py
└── .black
```

## Install

```bash
pip install -e .
```

## Run tests

```bash
pytest -v
```
